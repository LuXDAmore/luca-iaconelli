# Drink & D-Rank // App, Website, PWA

[DEMO](https://luxdamore.github.io/drink-and-drank)

## Installation

```bash

    # Clone the repo
    git clone https://github.com/LuXDAmore/drink-and-drank

    # Deps
    npm install

```

### Usage

```bash

    # Nuxt
    npm run ssr

    # Nativescript
    npm run android
    npm run ios

```
